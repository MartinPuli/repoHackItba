# M004 — Frontend, Integración y Demo

**Objetivo**: Conectar frontend con contratos y agente. Pulir UI. Preparar demo presentable para jurado.

**Deadline**: Horas 32-48 (final del hackathon)

## Slices

### S01 — Dashboard Principal
- Balance de Wallet y CajaFuerte desde contratos
- Gráfico de rendimientos DeFi
- Historial de transacciones
- Estado del agente en tiempo real

### S02 — Flujo de Onboarding
- Crear wallet via Factory (un click)
- Configurar herederos
- Tutorial interactivo del toggle de autonomía

### S03 — Panel del Agente
- Visualización de análisis del agente
- Feed de notificaciones del Co-Piloto
- Log de acciones autónomas
- Aprobación/rechazo de transacciones preparadas

### S04 — Operaciones DeFi UI
- Depositar/retirar de CajaFuerte
- Visualizar estrategia de yield spread
- Métricas: tasa Venus, rendimiento Rootstock, spread neto

### S05 — Demo & Pitch
- Script de demo con flujo completo
- Slides/deck de presentación
- Video de backup por si falla el live demo
- README completo del proyecto

## Criterio de Éxito
- Demo fluida de 5 minutos sin crashes
- Todos los flujos core funcionan end-to-end
- UI profesional y responsive
- Pitch claro del modelo de negocio
